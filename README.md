# HeatingReportMaker — генератор отчётов по отоплению

**HeatingReportMaker** — это десктоп-приложение для автоматизации расчётов и формирования отчётов по отоплению для жилых помещений. Программа читает данные из Excel-файла, выполняет расчёты и формирует два типа отчётов: подробный Excel-файл с таблицей и текстовое пояснение в Word.

Проект разработан для реального использования в управляющей компании и автоматизирует ручной труд сотрудника. Автоматизация сократила подготовку отчётов с 2–3 часов до 10 секунд.

---
## Возможности

- Чтение данных из Excel-файла
- Поиск данных по номеру квартиры
- Расчёт показателей:
  - общедомовое потребление (МОП)
  - индивидуальное потребление
  - итоговые суммы и объёмы в Гкал
  - промежуточные расчёты
- Генерация отчётов:
  - **Excel** — таблица с форматированием, цветами и итогами
  - **Word** — текстовое пояснение для жильца с детализацией расчётов. Шаблон для пояснений можно изменять под новые нужды, файл находится в папке Templates.
- Графический интерфейс на WPF, а также консольный интерфейс
- Интеграционные тесты на реальных данных. Тесты используют реальные Excel-файлы из папки TestData и проверяют корректность расчётов.
- Особенности
  - Входные данные могут содержать прочерки (-) — они интерпретируются как 0
  - Для квартир с расчётом по площади не будет создан отчёт, программа укажет причину
  - Расчёты выполняются с максимальной точностью, в отчётах значения округляются

- **C# / .NET 9**
- **WPF** — графический интерфейс
- **ClosedXML** — работа с Excel
- **DocX** — генерация Word-документов
- **xUnit** — тестирование
